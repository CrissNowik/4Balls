4Balls Game
====================

Description:
-------------
Popular, simple game for 2 players. Put 4 balls in one row - horizontal, vertical or oblique. Game is fully resizable and works fine on all mobile devices with webbrowser. 

Technology stack:
-------------
* HTML5,
* HTML5 Canvas,
* Java Script (ES6)
* CSS + SASS
